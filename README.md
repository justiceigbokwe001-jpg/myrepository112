# MVP Prototype (Next.js + Tailwind)

## Deploy to Vercel
1. Create a new GitHub repo and upload **these files (unzipped)** to the repo root.
2. Go to https://vercel.com/new → **Add New… > Project** → select the repo.
3. Confirm build command: `npm run build`. Click **Deploy**.
4. After it builds, click **Visit** on the latest deployment.

If you see a 404, ensure `app/page.tsx` is present at the repo root and deploy again.
